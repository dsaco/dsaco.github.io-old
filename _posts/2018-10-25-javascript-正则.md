###### 删除字符串str中的span标签
```
str.replace(/<\/?span[^>]*>/g,'');
```

##### 判断是否为手机号码
```
phoneNumber.match(/^1\d{10}$/);
```
##### 判断是否由中文组成
```
/^[\u4E00-\u9FA5]+$/.test('中文');
```
##### 判断是否为字母、数字、下划线组成
```
/^\w+$/.test(str);
```
##### 数字添加逗号
```
'12345678910'.replace(/\B(?=(\d{3})+(?!\d))/g, ',');
12,345,678,910
```
##### "Doe, John" 转换为 "John Doe"
```
"Doe, John".replace(/(\w+)\s*, \s*(\w+)/, "$2 $1");
```
##### 所有的花引号替换为直引号
```
`"a", "b"`.replace(/"([^"]*)"/g, `'$1'`);
```
